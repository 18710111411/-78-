## 📖 简介

李为东，前世行医四十年，孤老终身。
重生回到 1978 年春天，家徒四壁，两天后断粮。
带着道医传承，他决定改变命运——
让家人吃饱饭，让弟妹读书，阻止父亲早逝。

这是一个关于重生、医术、亲情和成长的故事。
